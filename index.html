Rent a Car API

C#, ASP.NET Core, Web API, Onion Architecture, CQRS & Mediator

View on GitHub

A comprehensive backend system built with ASP.NET Core following Onion architecture principles and using CQRS and Mediator patterns for a clean, scalable, and maintainable codebase.

Key Features:

  • Vehicle and Brand Management: Complete system for managing vehicle inventory including brand, model, images, mileage, fuel type, and transmission details
  • Location-based Services: Functionality for managing different office locations where customers can pick up and return vehicles
  • Dynamic Pricing Engine: Flexible module for defining and managing various pricing plans for vehicles (e.g., daily, weekly, monthly rates)
  • Content Management System: Includes a blog platform with author and category management to enhance customer engagement and share company news
  • Customer Feedback: Features for managing customer testimonials and handling contact messages sent through the platform
  • Statistical Reporting: API endpoint providing key business metrics such as total vehicle count, available locations, average rental prices, and most popular car brands

Architecture:

  • Presentation Layer (Web API): RESTful API built with ASP.NET Core Controllers that exposes all functionality to any client application
  • Application Layer: Implements core business logic using CQRS and Mediator patterns. This separates read and write operations for a more organized and efficient application flow
  • Data Access Layer: Manages all database interactions with Entity Framework Core
  • Domain Layer: Contains the core business entities that define the fundamental structure of the application

Bu proje, modern bir araç kiralama platformuna güç vermek için ASP.NET Core ile oluşturulmuş kapsamlı bir arka uç sistemidir. Temiz, ölçeklenebilir ve sürdürülebilir bir kod tabanı oluşturmak için Onion mimari prensiplerini takip eder ve CQRS ve Mediator desenlerini kullanır.

Temel Özellikler:

  • Araç ve Marka Yönetimi: Marka, model, resim, kilometre, yakıt tipi ve şanzıman gibi detayları içeren araç envanterini yönetmek için eksiksiz bir sistem
  • Konum Tabanlı Hizmetler: Müşterilerin araçları alıp iade edebileceği farklı ofis konumlarını yönetme işlevselliği
  • Dinamik Fiyatlandırma Motoru: Araçlar için çeşitli fiyatlandırma planlarını (örneğin günlük, haftalık, aylık ücretler) tanımlamak ve yönetmek için esnek bir modül
  • İçerik Yönetim Sistemi: Müşteri etkileşimini artırmak ve şirket haberlerini paylaşmak için yazar ve kategori yönetimine sahip bir blog platformu içerir
  • Müşteri Geri Bildirimi: Müşteri referanslarını yönetme ve platform üzerinden gönderilen iletişim mesajlarını ele alma özellikleri
  • İstatistiksel Raporlama: Toplam araç sayısı, mevcut konumlar, ortalama kiralama fiyatları ve en popüler araba markaları gibi temel iş metriklerini sağlayan bir API uç noktası

Mimari:

  • Sunum Katmanı (Web API): Tüm işlevleri herhangi bir istemci uygulamasına sunan, ASP.NET Core Denetleyicileri ile oluşturulmuş bir RESTful API
  • Uygulama Katmanı: CQRS ve Mediator desenlerini kullanarak temel iş mantığını uygular. Bu, okuma ve yazma işlemlerini ayırarak daha düzenli ve verimli bir uygulama akışı sağlar
  • Veri Erişim Katmanı: Entity Framework Core ile tüm veritabanı etkileşimlerini yönetir
  • Alan (Domain) Katmanı: Uygulamanın temel yapısını tanımlayan çekirdek iş varlıklarını içerir
